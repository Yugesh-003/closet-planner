# ClosetMind User Personas

## Overview
This document defines the user personas for ClosetMind, representing the primary target users of the application.

---

## Primary Persona: Priya - The Organized Professional

**Demographics:**
- Age: 25-35 years
- Occupation: Working professional (office job, hybrid/in-office work)
- Location: Urban area
- Tech Comfort: Moderate to high (comfortable with mobile apps and web applications)

**Primary Goals:**
- Efficiently manage wardrobe and plan outfits in advance
- Avoid outfit repetition and make better use of existing clothes
- Save time in morning routine by pre-planning outfits
- Ensure appropriate outfit choices for different occasions (work, events, casual)
- Receive timely reminders for important events requiring specific attire

**Key Characteristics:**
- Has a diverse wardrobe with clothes for multiple occasions (casual, formal, traditional)
- Attends various events (work meetings, social functions, family gatherings)
- Values organization and planning
- Wants to optimize wardrobe usage and reduce decision fatigue
- Appreciates weather-appropriate outfit suggestions

---

## Secondary Persona: Ananya - The Fashion-Conscious Student

**Demographics:**
- Age: 18-24 years
- Occupation: College student
- Location: Urban/semi-urban area
- Tech Comfort: High (digital native, frequent app user)

**Primary Goals:**
- Track outfit history to avoid repeating looks
- Get creative outfit suggestions for college and social events
- Manage wardrobe efficiently with limited clothing options
- Plan outfits for special occasions (interviews, presentations, parties)
- Stay organized with event-based outfit planning

**Key Characteristics:**
- Has a smaller but versatile wardrobe
- Frequently attends college events and social gatherings
- Values variety and creativity in outfit choices
- Budget-conscious, wants to maximize existing wardrobe
- Active social life requiring different outfit types

---

## Persona Usage in Stories
- **Priya** represents the primary target user for core features (wardrobe management, event planning, reminders)
- **Ananya** represents younger users focused on outfit variety and social events
- Most user stories will reference Priya as the primary persona
- Stories specific to student use cases or budget-conscious features may reference Ananya
